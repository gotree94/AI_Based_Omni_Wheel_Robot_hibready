## 1.3.1 학습모델 준비하기

(1) PC와 로봇을 Micro 5pin 케이블로 연결한다.


(2) 로봇의 전원 스위치를 ON 시킨다.


(3) PC의 "실행" 앱에서 아래와 같이 입력 하여 로봇에 접속한다.

```
\\192.168.55.1
```
